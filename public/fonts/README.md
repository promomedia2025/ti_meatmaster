# Fonts for PDF Generation

This directory should contain Roboto font files for proper Greek character support in PDF invoices.

## Required Font Files

Download the following Roboto font files and place them in this directory:

1. `Roboto-Regular.ttf` - Normal text
2. `Roboto-Medium.ttf` - Bold text
3. `Roboto-Italic.ttf` - Italic text
4. `Roboto-MediumItalic.ttf` - Bold italic text

## Download Links

You can download Roboto fonts from:
- Google Fonts: https://fonts.google.com/specimen/Roboto
- Or use: https://github.com/google/fonts/tree/main/apache/roboto

## Alternative: Use System Fonts

If you don't add custom fonts, the PDF will use system fonts which may not render Greek characters perfectly. For production use, it's recommended to add the Roboto fonts above.
